Nessus Scanning

To ensure that all security measure are to proper standard we used Nessus Vulnerability Scanner to evaluate the security of the network. We've performed penetration testing on the network the ensure that it is safe from malicious attacks. The examination of ports is done so that hackers can't exploit these ports and launch attacks that can be damaging to the network. This scan displays vulnerabilities that were found throughout the network and with these results we then applied the corresponding solutions. The first scan this procedure is launching a host discovery that displays full qualified domain name (FQDN), along with other details such as IP address and open ports. Upon discovery of the network, we then proceeded with the vulnerability scan.

Linux Firewall Configuring

I configure firewalls in Linux using tools like ufw for simplicity or iptables for more advanced configurations. I enable the firewall, allow essential connections like SSH or HTTP, set default policies for incoming and outgoing traffic, and ensure rules align with security requirements. I verify configurations with ufw status and make adjustments as needed. I prioritize security while ensuring necessary services remain accessible, contributing to robust network defense strategies.


Window Policy Configuring

In a Windows environment, end user policy outlines user responsibilities and behavior expectations, while password policy defines requirements for creating and managing passwords, aiming to enhance security. Group Policy enables administrators to centrally manage settings across a network, enforcing security policies, standardizing configurations, and ensuring compliance. These policies collectively contribute to maintaining security, integrity, and compliance within Windows environments.
